# Evil Invasion — UI 重设计 Spec (Infernal 风格)

> **日期**: 2026-05-16
> **状态**: 设计定稿，待实现
> **范围**: MainMenu, HeroPanel, HUD 三大核心 UI

---

## 1. 设计目标

将当前简陋的 Godot 默认控件 UI 升级为统一的 **Infernal（Diablo-like）红黑粗犷风**，提升玩家第一印象和沉浸感。

### 核心原则
- **统一 Theme 资源**：用 Godot Theme 统一管理所有 StyleBox、颜色、字体，替代各处散落的 hardcode 颜色
- **粗犷厚重**：粗边框 + 重阴影 + 纯色填充，不追求圆角和半透明
- **红黑金配色**：主色暗红 `#881100`→`#cc2200`，强调色橙金 `#ffaa44`，背景深黑 `#0a0806`
- **最大化游戏视野**：HUD 从 86px 压缩到 ~60px，去掉头像

- 纯 `Color(0.08, 0.08, 0.12)` 背景，无装饰
- 标题金色但无光晕/阴影

- **NEW GAME 按钮高亮**：与其余按钮区分，更亮的边框和填充

## 4. HeroPanel 重设计

### 4.1 当前问题
- 左右 45/55 分栏，属性区过大、技能树区偏小
- 用 ColorRect 堆叠做装饰，金色线只有 3px
- 技能按钮 46×62 偏小，开发者按钮暴露
- 属性行 28px 排列拥挤

### 4.2 新设计

**关键约束：HeroPanel 不能遮挡底部 HUD（~60px），HUD 在暂停时保持可见和可交互。**

├──────────────────────────────────────────────────┤ ← HeroPanel 底部边界

### 4.3 技术方案
- **分栏比例**：左侧属性 35%（~336px），右侧技能树 65%（~624px）
- **面板尺寸**：宽 960px，高 ≤ 900px（留底部 60px 给 HUD）
- **面板定位**：顶部对齐屏幕顶部，底部在 HUD 上方
- **半透明遮罩**：仅覆盖 HeroPanel 以上区域，不覆盖 HUD 区域
- **HUD 共存**：HeroPanel 打开时 HUD 保持显示，HUD 技能图标可点击设置快捷槽位
- **HUD 输入**：暂停时 HUD 使用 `PROCESS_MODE_ALWAYS` + `_input(event)` 处理点击
- **面板背景**：`Color(0.08, 0.06, 0.04, 0.95)`，边框 2px `#881100`
- **顶部/底部装饰线**：3px 渐变 `transparent → #cc4400 → transparent`
- **属性行**：32px 高，隔行微亮背景 `rgba(200,50,0,0.08)`
- **属性 [+] 按钮**：小型按钮 20×20，Infernal 风格
- **技能按钮**：升级到 52×52（从 46×62），元素色条加粗到 4px
- **技能按钮 hover**：边框变亮 `#cc4400`，微放大
- **技能按钮 pressed**：暗色填充
- **隐藏开发者按钮**：DevRelic/DevExp1000/DevExp5000 仅在 dev_mode 时显示
- **tooltip**：深色背景 + 暗红边框，与面板风格统一
- **属性区滚动**：当内容超出时 ScrollContainer 自动滚动

## 5. HUD 重设计

### 5.1 当前问题
- 底栏 86px 过高，头像用 `ground.png` 占位
- 技能图标 28×28 间距 2px，只有 8 个槽位但游戏有 18 个主动技能
- TipsLabel 文字硬编码且与当前快捷键不一致
- 快捷槽位 LMB/RMB/Shift/Space 布局紧凑

### 5.2 新设计

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│                   (游戏画面 874px)                     │
│                                                      │
│  [Buff 图标区域]                                       │
├──────────────────────────────────────────────────────┤
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ ← 2px 暗红顶线
│ ┌──────┐ ┌──────────────────────────────┐ ┌──┬──┐   │
│ │HP ██ │ │[1][2][3][4][5][6][7][8][9] │ │LMB│RMB│  │ ← 上排 22px 图标
│ │MP ██ │ │[10][11][12][13][14][15][16]│ │Shf│Spc│  │ ← 下排 22px 图标
│ └──────┘ │         [17][18]             │ └──┴──┘   │
│          └──────────────────────────────┘           │
│ ████████████████████████████░░░░░░░░░░░░░░░░░░░░░░ │ ← 8px EXP 条
│                    LEVEL 10                         │ ← 居中白色文字
└──────────────────────────────────────────────────────┘
          ← 总高约 60px (从 86px 压缩) →
```

### 5.3 技术方案

#### 整体结构
- **CanvasLayer**：layer=10（保持不变）
- **底栏高度**：从 86px 压缩到 ~60px（主区域 ~50px + EXP 条 8px）
- **游戏视口**：从 874px 增加到 ~900px

#### 左侧 HP/MP 区（~120px 宽）
- 两行：HP 条（6px 高）+ MP 条（6px 高）
- HP 条：`#331111` 背景 → `#cc2200` 填充
- MP 条：`#111133` 背景 → `#2244cc` 填充
- 右侧显示数值（8px 字号）
- **无头像、无等级文字**（等级在 EXP 条居中显示）

#### 中间技能栏
- **两排布局**：上排 9 个 + 下排 9 个 = 18 个主动技能
- **图标尺寸**：22×22px（原 28×28），间距 3px
- **被动技能不显示**：fortuna, telekinesis, stone_enchanted 共 3 个
- **快捷键标注**：每个图标右下角显示默认快捷键（5px 字号 `#886644`）
- **上排快捷键**：LMB, RMB, Z, X, C, 2, 3, 4, Q
- **下排快捷键**：R, E, U, F, G, H, B, N, O
- **未学习技能**：灰色半透明 + 灰色边框
- **冷却遮罩**：保持现有 `cooldown_overlay.gd` 扇形遮罩
- **自动释放指示**：保持现有虚线光圈

#### 右侧快捷槽位（2×2 网格）
- **布局**：上排 LMB/RMB，下排 Shift/Space
- **槽位尺寸**：36×22px
- **边框**：1px `#aa4422`
- **填充**：`rgba(200,50,0,0.15)`
- **文字**：7px `#ffaa44`

#### 底部 EXP 条（8px 高）
- **全宽**：横跨整个 1024px
- **背景**：`#110400`
- **填充**：`#881100` → `#cc4400` 渐变
- **居中文字**："LEVEL X"，8px 白色 `#ffcc88`，粗体，文字阴影 `1px 1px 0 #000`

#### 配色
- **底栏背景**：`linear-gradient(rgba(40,10,5,0.92), rgba(15,3,1,0.96))`
- **顶部分割线**：2px `#881100`

#### 移除内容
- ❌ 英雄头像（PortraitBg + HeroPortrait）
- ❌ TipsLabel（快捷键已在图标上标注，不再需要底部提示文字）
- ❌ HP/MP 百分比标签
- ❌ 左侧等级文字

## 6. Theme 资源方案

创建统一 Theme 文件 `res://Art/Theme/infernal_theme.tres`：

```gdscript

# - panel_inner: bg=rgba(20,8,8,0.95), corner=0
#
# 颜色常量：
# - font_color: #ffaa44
# - font_color_dim: #886644
# - font_color_title: #dd2200
#
# 字体设置：
# - 默认字体大小 11px
# - 标题字体大小 38px / 32px
```

### Theme 应用范围
- MainMenu
- GameModeSelect
- HeroPanel
- PauseMenu（部分覆盖）
- LevelSelect
- Options
- HighScores
- Credits

## 9. 分辨率适配

- 目标分辨率：**1024×960**（宽×高）
- HUD 底栏宽度：1024px（全宽）
- 技能栏总宽：9×22 + 8×3 = 222px（上排），在 1024px 中占比 22%，余量充足
- HUD 总高 ~60px，游戏视口 ~900px
